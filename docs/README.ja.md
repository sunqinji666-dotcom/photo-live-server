# Photo Live Server

A self-hosted photo event delivery service for uploads, galleries, and live updates.

![プロジェクトのビジュアル](docs/assets/hero-system-v1.png)

## できること

A self-hosted photo event delivery service for uploads, galleries, and live updates. プロジェクトの README にある手順で、環境に合わせて実行してください。

## はじめ方

認証情報や個人データはリポジトリに保存せず、環境変数または Git で無視されるローカル設定を使います。
