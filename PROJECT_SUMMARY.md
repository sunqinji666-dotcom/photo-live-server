## 🚀 快速开始

### 方式一：本地测试

```bash
cd /Users/jacksun/photo-live-server

# 1. 创建数据库
mysql -u root -p < database/init.sql
mysql -u root -p < database/seed.sql

# 2. 配置环境
cp .env.example .env
# 编辑 .env 填写数据库配置

# 3. 启动
chmod +x start.sh
./start.sh
```

访问：http://localhost:3000

### 方式二：宝塔部署

```bash
# 1. 上传项目到服务器
cd /www/wwwroot/your-domain

# 2. 运行部署脚本
chmod +x deploy.sh
./deploy.sh

# 3. 在宝塔面板配置
# - 创建数据库并导入 init.sql
# - 创建 Node.js 项目
# - 配置反向代理
# - 配置 SSL（推荐）
```

详细步骤查看：[DEPLOY.md](DEPLOY.md)

## 🎯 核心技术栈

### 后端
- **框架**：Express.js 5.x
- **数据库**：MySQL 5.7+
- **认证**：JWT + bcrypt
- **图片处理**：Sharp
- **文件上传**：Multer
- **云存储**：七牛云 SDK
- **实时通信**：Socket.IO

### 前端
- **纯原生**：HTML5 + CSS3 + JavaScript
- **无需框架**：零依赖，轻量高效
- **响应式**：完美适配桌面/平板/手机

### 部署
- **进程管理**：PM2
- **反向代理**：Nginx
- **部署工具**：宝塔面板
